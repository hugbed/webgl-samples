# WebGL Samples

## Setup

### Install

To run this sample, you need [npm](https://www.npmjs.com/get-npm).

Then from the root folder, run this command to install dependencies:

```
npm install
```

### Run

Start the server (see below) and navigate to http://localhost:8000

```
npm run dev
```
